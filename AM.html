Rozwiązanie:

Podzielmy równanie na składniki:

\( y' = \frac{e^x (y + 1)}{e^x + 1} \)

Przekształćmy równanie:

\( \frac{1}{y + 1} \frac{dy}{dx} = \frac{e^x}{e^x + 1} \)

Rozdzielmy zmienne:

\( \frac{1}{y + 1} dy = \frac{e^x}{e^x + 1} dx \)

Zintegrujmy obie strony:

\( \int \frac{1}{y + 1} dy = \int \frac{e^x}{e^x + 1} dx \)

Lewa strona:

\( \int \frac{1}{y + 1} dy = \ln|y + 1| \)

Prawa strona:

Użyjmy podstawienia \( u = e^x + 1 \), wtedy \( du = e^x dx \):

\( \int \frac{e^x}{e^x + 1} dx = \int \frac{1}{u} du = \ln|u| = \ln|e^x + 1| \)

Zatem:

\( \ln|y + 1| = \ln|e^x + 1| + C \)

Zastosujmy warunek początkowy \( y(0) = -4 \):

\( \ln|-4 + 1| = \ln|e^0 + 1| + C \)

\( \ln 3 = \ln 2 + C \)

\( C = \ln 3 - \ln 2 = \ln \frac{3}{2} \)

Ostateczne rozwiązanie:

\( \ln|y + 1| = \ln \left( \frac{3}{2} (e^x + 1) \right) \)

\( y + 1 = \frac{3}{2} (e^x + 1) \)

\( y = \frac{3}{2} e^x + \frac{3}{2} - 1 \)

\( y = \frac{3}{2} e^x + \frac{1}{2} \)

Równanie różniczkowe:

\( y' - y \cot x = \cos x \sin^2 x \)

Rozwiązanie:

Przekształcenie równania:

\( y' + P(x)y = Q(x) \)

Gdzie \( P(x) = -\cot x \) i \( Q(x) = \cos x \sin^2 x \).

Czynnik całkujący:

\( \mu(x) = e^{\int P(x) dx} = e^{\int -\cot x dx} = e^{-\ln|\sin x|} = \frac{1}{\sin x} \)

Mnożymy obie strony równania przez czynnik całkujący:

\( \frac{1}{\sin x} y' - \frac{1}{\sin x} y \cot x = \frac{\cos x \sin^2 x}{\sin x} \)

\( \frac{d}{dx} \left( \frac{y}{\sin x} \right) = \cos x \sin x \)

Integrujemy obie strony:

\( \frac{y}{\sin x} = \int \cos x \sin x dx \)

Używamy podstawienia \( u = \sin x \), wtedy \( du = \cos x dx \):

\( \int \cos x \sin x dx = \int u du = \frac{u^2}{2} = \frac{\sin^2 x}{2} \)

Zatem:

\( \frac{y}{\sin x} = \frac{\sin^2 x}{2} + C \)

\( y = \frac{\sin^3 x}{2} + C \sin x \)
